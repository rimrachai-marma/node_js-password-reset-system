# Node JS Password Reset System
Node JS password reset system and send email using Nodemailer and Google OAuth2


### Env Variables

Create a .env file in then root and add the following

```
PORT =
MONGO_URL =
JWT_SECRET =
OAUTH_CLIENTID =
OAUTH_CLIENT_SECRET =
OAUTH_REFRESH_TOKEN = 
MAIL_USERNAME = 
MAIL_PASSWORD = 
EMAIL = 
CLIENT_URL = 
```
